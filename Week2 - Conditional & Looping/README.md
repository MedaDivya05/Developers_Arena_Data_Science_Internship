# Week 2 - Student Grade Calculator

## Project Overview
This project is part of **Week 2: Making Decisions & Repeating Tasks in Python** for the **Data Science Online Internship** at **The Developers Arena**.

The objective of this project is to build a **Student Grade Calculator** using Python that accepts a student's name and marks, validates the input, calculates the grade using conditional statements, and displays a personalized result with encouraging feedback.

This project focuses on core Python concepts such as:
- `if-elif-else` statements
- Comparison operators
- `while` loops
- Functions
- Input validation
- Basic error handling using `try-except`

---

## 🎯 Project Goals and Objectives
- Understand decision-making in Python using `if-elif-else`
- Implement grading logic based on marks
- Validate user input within the range 0–100
- Use a `while` loop to repeatedly ask for valid input
- Create reusable functions for cleaner code
- Display encouraging messages based on grade
- Practice real-world problem solving with Python

---

## 🛠️ Technologies Used
- **Python 3.x**
- Built-in Python concepts:
  - `input()`
  - `print()`
  - `if-elif-else`
  - `while` loop
  - Functions
  - `try-except`
