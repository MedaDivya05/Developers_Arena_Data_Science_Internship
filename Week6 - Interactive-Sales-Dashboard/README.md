# 📊 Week 6 - Interactive Sales Dashboard

## 📌 Project Overview

This project was completed as part of **Week 6: Data Visualization Mastery with Seaborn** for the **Data Science Online Internship** at **The Developers Arena**.

The objective of this project is to create a professional and interactive sales dashboard using Python visualization libraries such as **Seaborn**, **Matplotlib**, and **Plotly**. The project focuses on transforming raw sales data into meaningful visual insights through statistical plots, trend analysis, correlation analysis, and interactive charts.

The dashboard helps analyze:

- Product-wise sales performance
- Regional sales contribution
- Price distribution
- Monthly sales trends
- Correlation between numerical variables
- Statistical distribution patterns

This project demonstrates how data visualization can help businesses make informed decisions by presenting complex data in an easy-to-understand visual format.

---

# 🎯 Project Goals and Objectives

The major objectives of this project are:

- Learn advanced data visualization techniques
- Create professional statistical visualizations using Seaborn
- Build interactive charts using Plotly
