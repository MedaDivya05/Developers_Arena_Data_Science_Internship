# 📊 Week 5 - Customer Sales Analysis

## 📌 Project Overview

This project was completed as part of **Week 5: Advanced Data Manipulation with Pandas** for the **Data Science Online Internship** at **The Developers Arena**.

The objective of this project is to perform advanced customer and sales analysis using Python and the Pandas library. The project combines customer churn data with sales transaction data to identify customer purchasing behavior, analyze sales trends, create advanced summaries, and generate business insights.

The project demonstrates a complete data analysis workflow including:

- Loading and exploring datasets
- Cleaning and preprocessing data
- Working with datetime values
- Filtering data using multiple conditions
- Grouping and aggregation operations
- Merging multiple datasets
- Creating pivot tables
- Generating professional visualizations
- Writing business insights and recommendations

This project helped in understanding how advanced Pandas operations are used in real-world business analytics and customer behavior analysis.
