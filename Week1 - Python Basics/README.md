# Week 1 - Python Basics: Personal Introduction Program

## 📌 Project Overview
This project is part of **Week 1: Python Basics** for the **Data Science Online Internship** at **The Developers Arena**.

The objective of this project is to understand the fundamentals of Python programming by creating a simple interactive program that asks the user for personal information and displays a friendly personalized introduction message.

This project helped in learning the basic building blocks of Python such as:
- Variables
- User input using `input()`
- Output using `print()`
- Strings and formatting
- f-strings
- Basic program flow

---

## 🎯 Project Goal
To create a beginner-friendly Python program that:
- Asks the user for personal details
- Stores the details in variables
- Displays a customized welcome message
- Uses at least 3 input questions
- Produces a friendly and welcoming output

---

## 🛠️ Technologies Used
- **Python 3.x**
- Built-in Python functions:
  - `input()`
  - `print()`
  - Variables
  - f-strings
